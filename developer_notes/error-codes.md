# Error Codes

| Code | Name | Description |
|------|------|-------------|
| 1 | NotInitialized | Contract not yet initialised |
| 2 | AlreadyInitialized | Init called more than once |
| 3 | Unauthorized | Caller lacks required authority |
| 4 | InsufficientBalance | Not enough tokens |
| 5 | InvalidAmount | Amount must be positive |
| 6 | ContractPaused | Deposits/distributions are paused |
