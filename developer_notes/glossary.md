# Glossary

**WASM** – WebAssembly, the compile target for Soroban contracts.

**Ledger** – A single state snapshot of the Stellar network.

**TTL** – Time-To-Live; how many ledgers a storage entry persists.

**SAC** – Stellar Asset Contract; the canonical token wrapper.

**XDR** – External Data Representation; Stellar's binary serialisation format.
