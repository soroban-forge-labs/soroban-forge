# hello-forge

A minimal Soroban smart contract, generated by
[soroban-forge](https://github.com/soroban-forge-labs/soroban-forge) from the
`hello-world` template.

## Develop

```sh
cargo test                 # run unit tests
stellar contract build     # build the deployable wasm
```

## Deploy to testnet

```sh
stellar contract deploy \
  --wasm target/wasm32v1-none/release/hello_forge.wasm \
  --source-account <your-identity> \
  --network testnet
```

See the [Stellar docs](https://developers.stellar.org/docs/build/smart-contracts)
for the full workflow.
